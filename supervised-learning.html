Supervised learning

Introduction¶

This section will cover the details of implementing the fit function above: That is, how to use a dataset of labelled samples (x1,y1),…,(xN,yN)(x_1, y_1), \dots, (x_N, y_N)(x1​,y1​),…,(xN​,yN​) to find a function fff that minimizes the empirical risk. This requires two ingredients:

  1. A function class F\mathcal{F}F to search over
  2. A fitting method for minimizing the empirical risk over this class

The two main function classes we will cover are linear models and neural networks. Both of these function classes are parameterized by some parameters θ, and the fitting method will search over these parameters to minimize the empirical risk:

Definition 1 (Parameterized empirical risk minimization)

Given a dataset of samples (x1,y1),…,(xN,yN)(x_1, y_1), \dots, (x_N, y_N)(x1​,y1​),…,(xN​,yN​) and a class of functions F\mathcal{F}F parameterized by θ, we to find a parameter (vector) θ^\hat \thetaθ^ that minimizes the empirical risk:

θ^=arg⁡min⁡θ1N∑i=1N(yi−fθ(xi))2\hat \theta = \arg\min_{\theta} \frac{1}{N} \sum_{i=1}^N (y_i - f_\theta(x_i))^2θ^=argθmin​N1​i=1∑N​(yi​−fθ​(xi​))2
(1)

The most common fitting method for parameterized models is gradient descent.

Definition 2 (Gradient descent)

Letting L(θ)∈RL(\theta) \in \mathbb{R}L(θ)∈R denote the empirical risk in terms of the parameters, the gradient descent algorithm updates the parameters according to the rule

θt+1=θt−η∇θL(θt)\theta^{t+1} = \theta^t - \eta \nabla_\theta L(\theta^t)θt+1=θt−η∇θ​L(θt)
(2)

where η>0\eta > 0η>0 is the learning rate.

from jaxtyping import Float, Array
from collections.abc import Callable
Params = Float[Array, " D"]


def gradient_descent(
    loss: Callable[[Params], float],
    θ_init: Params,
    η: float,
    epochs: int,
):
    """
    Run gradient descent to minimize the given loss function
    (expressed in terms of the parameters).
    """
    θ = θ_init
    for _ in range(epochs):
        θ = θ - η * grad(loss)(θ)
    return θ

Linear regression¶

In linear regression, we assume that the function fff is linear in the parameters:

F={x↦θ⊤x∣θ∈RD}\mathcal{F} = \{ x \mapsto \theta^\top x \mid \theta \in \mathbb{R}^D \}F={x↦θ⊤x∣θ∈RD}
(3)

This function class is extremely simple and only contains linear functions. To expand its expressivity, we can transform the input xxx using some feature function ϕ, i.e. x~=ϕ(x)\widetilde x = \phi(x)x=ϕ(x), and then fit a linear model in the transformed space instead.

def fit_linear(X: Float[Array, "N D"], y: Float[Array, " N"], φ=lambda x: x):
    """Fit a linear model to the given dataset using ordinary least squares."""
    X = vmap(φ)(X)
    θ = np.linalg.lstsq(X, y, rcond=None)[0]
    return lambda x: np.dot(φ(x), θ)

Neural networks¶

In neural networks, we assume that the function fff is a composition of linear functions (represented by matrices WiW_iWi​) and non-linear activation functions (denoted by σ):

F={x↦σ(WLσ(WL−1…σ(W1x+b1)⋯+bL−1)+bL)}\mathcal{F} = \{ x \mapsto \sigma(W_L \sigma(W_{L-1} \dots \sigma(W_1 x + b_1) \dots + b_{L-1}) + b_L) \}F={x↦σ(WL​σ(WL−1​…σ(W1​x+b1​)⋯+bL−1​)+bL​)}
(4)

where Wi∈RDi+1×DiW_i \in \mathbb{R}^{D_{i+1} \times D_i}Wi​∈RDi+1​×Di​ and bi∈RDi+1b_i \in \mathbb{R}^{D_{i+1}}bi​∈RDi+1​ are the parameters of the iii-th layer, and σ is the activation function.

This function class is much more expressive and contains many more parameters. This makes it more susceptible to overfitting on smaller datasets, but also allows it to represent more complex functions. In practice, however, neural networks exhibit interesting phenomena during training, and are often able to generalize well even with many parameters.

Another reason for their popularity is the efficient backpropagation algorithm for computing the gradient of the empirical risk with respect to the parameters. Essentially, the hierarchical structure of the neural network, i.e. computing the output of the network as a composition of functions, allows us to use the chain rule to compute the gradient of the output with respect to the parameters of each layer.

Nielsen (2015) provides a comprehensive introduction to neural networks and backpropagation.
